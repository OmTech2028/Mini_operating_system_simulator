#  Mini Operating System Simulator

A **Java-based** simulation of core OS concepts:
- **Process Scheduling**: FCFS & Round Robin
- **Memory Management**: First-Fit allocation with splitting

##  Features
-  Interactive menu-driven interface
-  Visual Gantt charts
-  Real-time metrics (Waiting time, Turnaround time)
-  Dynamic memory allocation
-  Custom process input
-  Professional console output
